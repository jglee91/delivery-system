# Delivery Admin

## Git branch

> feature/admin

## Set up

```bash
# init
$ yarn create next-app delivery-admin --typescript

$ cd delivery-admin

## add dependencies
$ yarn add @material-ui/core @material-ui/icons styled-components
$ yarn add -D @types/styled-components
```
